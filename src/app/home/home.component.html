MC-Form-Grid

An Angular NPM Package for organizing Reactive Forms into easy layout grids.

Made to provide an easy way to lay out forms with perfectly aligned labels and columns without having to write the CSS yourself. Check out each component's documenation above, or scroll down to see more examples.

Install using npm i mc-form-grid --save or visit the NPM page.

Then import MCFormGridModule in the Angular Module you want to use it in.

import  {{ '{' }} McFormGridModule {{ '}' }} from 'mc-form-grid';

@NgModule({{ '{' }}
  declarations: [
  ],
  imports: [
    McFormGridModule,
 ],

See the GitHub Repo Here to help contribute!

In the Component:

    <mc-form-grid columns="2">
      <mc-form-input label="Input 1">
        <input type="text" placeholder="I take up column 1">
      </mc-form-input>
      <mc-form-input label="Input 2">
        <input type="text" placeholder="I take up column 2">
      </mc-form-input>
      <mc-form-input label="Textarea 1" columnSpan="2">
        <textarea placeholder="I take up 2 columns!"></textarea>
      </mc-form-input>
      <mc-form-input label="Textarea 2" rowSpan="2">
        <textarea rows="3" placeholder="I take up 2 rows!"></textarea>
      </mc-form-input>
      <mc-form-input label="Select 1" startingCol="2">
        <select>
          <option>Select 1</option>
        </select>
      </mc-form-input>
      <mc-form-input label="Select 2" startingCol="2">
        <select>
          <option>Select 2</option>
        </select>
      </mc-form-input>
    </mc-form-grid>
Set your desired number of columns:
<mc-form-grid columns="4">
 <mc-form-input *ngFor="let a of [1,2,3,4,5,6,7,8]" [label]="'Input ' + a">
   <input type="text">
 </mc-form-input>
</mc-form-grid>

Setting the number of columns will create ([number of columns] * 2) columns. In the above case, there will be 4 columns for labels and 4 columns for the inputs.

Label columns will expand to the width of the largest label in that column:

<mc-form-grid columns="3">
  <mc-form-input label="Extremely Long Label Here">
    <input type="text">
  </mc-form-input>
  <mc-form-input *ngFor="let a of [1,2,3,4,5]" [label]="'Input ' + a">
    <input type="text">
  </mc-form-input>
</mc-form-grid>

Labels can be constrained or lengthend by setting the following properties on mc-form-grid:

labelWidth
maxLabelWidth
labelPadding

Labels can also align left or right by setting:

textAlignLeft='true | false'

Add a divider to split up your Form Grid into sections.
<mc-form-grid columns="2">
  <mc-form-input label="Input 1" columnSpan="2">
    <input type="text">
  </mc-form-input>
  <mc-form-divider label="Divider With Label"></mc-form-divider>
  <mc-form-input label="Input 2" >
    <input type="text" placeholder="Divider with Label Above">
  </mc-form-input>
  <mc-form-input label="Input 3" >
    <input type="text" placeholder="Divider without Label Below">
  </mc-form-input>
  <mc-form-divider></mc-form-divider>
  <mc-form-input label="Input 4">
    <input type="text">
  </mc-form-input>
  <mc-form-input label="Input 5">
    <input type="text">
  </mc-form-input>
</mc-form-grid>
