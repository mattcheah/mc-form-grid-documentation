MC-Form-Grid

An Angular NPM Package for organizing Reactive Forms into easy layout grids.

Made to provide an easy way to lay out forms with perfectly aligned labels and columns without having to write the CSS yourself. Check out the full list of features, or scroll down to see more examples.

In the Component:

    <mc-form-grid columns="2">
      <mc-form-input label="Input 1">
        <input class="form-control" type="text" placeholder="I take up column 1">
      </mc-form-input>
      <mc-form-input label="Input 2">
        <input class="form-control" type="text" placeholder="I take up column 2">
      </mc-form-input>
      <mc-form-divider label="My Divider Label"></mc-form-divider>
      <mc-form-input label="Textarea 1" columnSpan="2">
        <textarea class="form-control" placeholder="I take up 2 columns!"></textarea>
      </mc-form-input>
      <mc-form-input label="Textarea 2" rowSpan="2">
        <textarea class="form-control" rows="3" placeholder="I take up 2 rows!"></textarea>
      </mc-form-input>
      <mc-form-input label="Select 1" startingCol="2">
        <select class="form-control">
          <option>Select 1</option>
        </select>
      </mc-form-input>
      <mc-form-input label="Select 2" startingCol="2">
        <select class="form-control">
          <option>Select 2</option>
        </select>
      </mc-form-input>
    </mc-form-grid>
