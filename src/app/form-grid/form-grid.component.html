Form Grid Component

mc-form-grid

Component Input Directives:

Directive Type Description
labelWidth string Specify a hard width for all labels. Not recommended to use this.
labelPadding string
boldLabel boolean Automatically makes the label bold.
appendColon boolean Appends a colon to every form-input and form-custom label. Labels with asterisks as their last character will have it moved to the outside of the colon.
textAlignLeft boolean Defaults to aligning label text to the left, but if true the label is aligned right.
maxLabelWidth boolean If label width exceeds this length, wrap the label.
errorHighlightOnTouched boolean Toggles whether error highlights show on invalid form inputs after they've been touched.
False means no error highlights show unless `errorHighlight` is true
errorHighlight boolean Toggles error highlights on/off for invalid form inputs. Automatically and always false if `errorHighlightOnTouched` is true. Otherwise, can be toggled on and off by parent component.
This is set to true by default even though it will be set to false on NgOnInit(). This is a fallback just in case `errorHighlightOnTouched` is false. In that case, the errors will show up immediately (as opposed to never showing up if `errorHighlight` is not set.)
@example <mc-form-grid [errorHighlight]="formHasBeenSubmitted"> - only displays errors after a form has been submitted and not before.
