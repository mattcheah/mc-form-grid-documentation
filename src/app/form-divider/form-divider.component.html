Form Divider

mc-form-input

A Form Divider is just a simple line with top/bottom margin that separates sections of a form.

Passing in a [label] will add that label in the middle of the line. You can also optionally set the width of the line and the top/bottom margin

Component Directives:

Directive Type Description
label string The label that appears in the middle of the divider
width string Default width of 100% of the Form Grid width, but can be changed to be smaller
margin string A value to set the top and bottom margin property. Defaults to 10px.
